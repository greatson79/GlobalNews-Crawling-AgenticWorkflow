# 문체 가이드 (Style Guide)

> 최윤식 박사의 통찰보고서 문체를 재현하기 위한 구체적 규칙.

## 1. 서술 톤 (Narrative Tone)

### 전문가 분석 톤

| 사용 | 회피 |
|------|------|
| "~로 판단된다" | "~인 것 같다" |
| "~을 시사한다" | "~일지도 모른다" |
| "~로 해석된다" | "~라고 생각된다" |
| "~로 전망된다" | "~가 될 수도 있다" |
| "~로 규정했다" | "~라고 했다" |
| "~을 예고하고 있다" | "~이 걱정된다" |

### 어미 패턴

**분석 서술**: ~이다, ~되었다, ~한다, ~있다 (단정적 평서문)
```
현재 이란 주변의 군사적 긴장은 이라크 전쟁 이후 최고조에 달해 있으며,
단순한 위협 수준을 넘어 실전 돌입 직전의 징후들이 포착되고 있다.
```

**판단 서술**: ~로 판단된다, ~로 해석된다, ~을 시사한다
```
이는 완전한 제거가 아닌 협상의 여지를 열어둔 '통제된 후퇴'로 해석된다.
```

**전망 서술**: ~으로 전망된다, ~것으로 보인다, ~할 것이다
```
리스크는 하반기에 정점에 달할 것으로 예측된다.
```

## 2. 전문용어 처리

### 영문 병기 규칙

**첫 출현 시** 반드시 영문 병기. 이후는 한국어만 사용 가능.

```
✅ 사모신용(Private Credit) 시장은...  → 이후: 사모신용 시장은...
✅ 청산 캐스케이드(Liquidation Cascade) 메커니즘
✅ 즉각 대응 태세(Ready-to-Strike)
✅ 조기경보 지표(Early Warning Signals)
```

### 한자 병기

의미 강화가 필요한 경우에만:
```
사궁(死宮)  — 정치적 막다른 골목의 심각성 강조
사건(事件)  — 문서 제목에서의 격식
```

### 기관/인명

**기관**: 약칭(풀네임) 순서
```
OCC(통화감독청), SNSC(국가안전보장회의), ICE(이민세관집행국)
```

**인명**: 한국어명 + 영문명(최초 1회)
```
모하메드 엘 에리언(Mohamed El-Erian) 박사
케빈 워시(Kevin Warsh)
```

## 3. 문단 구조

### 볼드 소제목 패턴

단락 내에서 **주제 전환** 시 볼드 소제목을 사용:

```markdown
**국내 정치적 사궁(死宮):**
지지율 급락과 악재: 앱스타인 사건 명단에 하워드 러트닉 등 측근이 포함된 점과...

**정치적 데드라인:**
11월 중간 선거를 앞두고 가시적인 성과가 절실하며...

**관리된 긴장과 '애드벌룬' 전술:**
'90% 전쟁 확률' 보도는 이란을 극도로 압박해...
```

### 콜론(:) 뒤 서술 패턴

볼드 소제목 뒤에 콜론(:)으로 연결하여 바로 서술 시작:

```markdown
**전략 자산의 압도적 전개:** 미 제럴드 포드호를 포함한 2개의 항공모함 전단과...
**하메네이의 4단계 승계 특명:** 아야톨라 하메네이는 지도부 암살 및 유고에 대비해...
```

### 데이터 임베딩

수치 데이터를 문장 흐름 안에 자연스럽게 삽입:

```
✅ "이는 미 해군 전력의 약 1/2, 전체 해군력의 1/3이 집중된 규모로,
    투입된 자산 가치만 수천억 달러(수백조원)에 달하는 천문학적인 규모다."

✅ "1.8조 달러 규모의 사모신용 시장은 2025년 연말부터 이미 비선형적
    리스크(Non-linear Risk) 징후를 보였다."

❌ "미 해군이 많은 전력을 배치했다." (구체성 없음)
```

## 4. 특수 표현 패턴

### 은유/비유 활용

최윤식 박사 특유의 비유적 표현:

| 비유 | 의미 | 출처 |
|------|------|------|
| "탄광 속의 카나리아" | 위기의 선행 지표 | 사모신용 분석 |
| "블랙박스 중국" | 정보 불투명 체제 | 태자당 분석 |
| "유리 상자 속의 투명한 중국" vs "블랙박스" | 외부 관찰 가능 vs 불가능 | 태자당 결론 |
| "외과 수술적 공습" | 정밀 제한 타격 | 이란 군사 시나리오 |
| "'미친개' 전략" | 예측 불가능성을 무기로 사용 | 트럼프 협상 전술 |
| "상처 입은 독재자" | 내부 위기가 외부 공격성을 유발 | 시진핑 분석 |

### 경고/강조 패턴

```
"핵심 관전 포인트:" — 시나리오 분기 조건 제시
"조기경보 지표:" — 모니터링 대상 나열
"현 정세 규정:" — 현재 상태의 성격 규정
"결론:" — 종합 판단
```

## 5. 표 사용 규칙

PDF 내 표 사용 패턴:

| 용도 | 예시 |
|------|------|
| 법안/정책 비교 | 가상자산 관련 주요 법안의 교착 상태 (p.13) |
| 시나리오 비교 | 복수 시나리오의 조건·확률·결과 |
| 지표 대시보드 | 조기경보 지표 + 현재 상태 + 임계값 |

표 내부에서도 구체적 설명을 포함 (빈 셀 지양):

```markdown
| 법안명 | 핵심 내용 및 현재 상태 (소스 기준) |
|--------|----------------------------------|
| 루미스법(Lummis Bill) | 5년간 비트코인 100만 개 전략 비축 추진. 현재 국가 부채 문제와 국회 통과 난항으로 동력을 상실한 상태. |
```

## 6. 직접 인용 규칙

### 원문 영어 병기 (필수)

타인의 발언을 인용할 때 반드시 **원문 영어를 괄호 안에 병기**한다. 한국어 의역만으로 직접 인용 부호(`""`)를 사용하면 원문 왜곡 위험이 있다.

```
✅ 슈머는 "지상군 배치를 하게 될까 그 어느 때보다 두렵다
   (I fear now more than ever that we are going to put boots on the ground)"고 우려했다.

✅ 구테흐스는 "외교 기회를 허비했다(squandered a chance for diplomacy)"고 비판했다.

❌ 슈머는 "우리는 지상군을 투입하게 될 것"이라고 경고했다.  (의역을 직접 인용으로 표기)
❌ 구테흐스는 "외교적 기회를 낭비했다"고 비판했다.  (원문 미확인 의역)
```

### 간접 인용 vs 직접 인용 구분

| 유형 | 표기 | 예시 |
|------|------|------|
| 직접 인용 | 큰따옴표 + 영문 원문 | "외교 기회를 허비했다(squandered a chance for diplomacy)" |
| 간접 인용 | 따옴표 없이 서술 | 구테흐스 사무총장은 외교적 기회가 낭비되었다고 비판했다 |
| 매체 표현 인용 | 작은따옴표 | CNN은 '정치적 악몽'이라 표현했다 |

## 7. 진행 중 사건의 서술 규칙

### 확정 vs 유보 표현

보고서 작성일 **당일**에 발생 중이거나 수치가 확정되지 않은 사건:

| 상황 | 확정 표현 (금지) | 유보 표현 (사용) |
|------|----------------|----------------|
| 당일 시장 데이터 | "KOSPI 10%대 급반등" | "KOSPI 반등 전망" |
| 진행 중 군사 작전 | "500명 사망" | "500명 이상 사망한 것으로 보도됨" |
| 미확인 보도 | "CIA가 무장 지원" | "CIA 무장 지원설이 전해졌다" |

### 한국어 전용 소스 표기

영어 소스로 교차 검증이 불가능한 한국 언론 보도:

```
✅ 조선일보·한겨레·연합뉴스가 동시에 보도했다 (한국 언론 보도 기준).
✅ 한겨레 보도에 따르면 (~로 보도됨).

❌ 쿠르드족 수천 명이 이란 국경을 넘었다.  (단정 — 한국 언론만 보도한 내용)
```

## 8. 사상자·피해 수치 표기

### 포함 범위 명시 (필수)

사상자 수를 인용할 때 반드시 포함 범위를 명시한다:

```
✅ "165명 이상(7~12세 소녀 및 교직원 포함)이 사망했다"
✅ "이란 민간인 1,000명 이상(초등학교 참사 포함) 사망"

❌ "소녀 165~175명이 사망했다"  (교직원 누락 — 총 사망자와 구성원 혼동)
❌ "1,000명 사망"  (군인/민간인 구분 없음)
```

## 9. 문장 길이와 리듬

- **분석 문장**: 20-40자 (정보 밀도 높게)
- **연결 문장**: 10-20자 (흐름 연결)
- **강조 문장**: 짧고 단정적

```
이는 시스템의 자기 교정 능력 상실을 의미한다.   ← 강조 (짧고 단정)
1인 독재 체제의 완성은 중국 공산당 시스템을     ← 분석 (정보 밀도)
'오류 수정 불가능 상태'로 몰아넣었다.
이제 중국 공산당은 '제도가 사람을 통제하는       ← 연결→전환
시스템'에서 '사람이 제도를 파괴하는 상태'로
전락했다.
```
