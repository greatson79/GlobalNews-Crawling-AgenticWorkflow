## 헤더/푸터 규칙

| 요소 | 규칙 |
|------|------|
| 배너 | `━━━` 라인으로 감싼 "미래학자 최윤식 박사의 미래통찰" |
| 날짜 | "YYYY년 M월 D일, 존경하는 독자들에게..." |
| 발간 기준 | 고정 문구 (위 템플릿 참조) |
| 저자 역량 | 2012년부터의 예측 실적 강조 |
| 메인 타이틀 | 영문 포함 가능 (예: "Watching Crisis: 몇 가지 단상들") |
| 토픽 소개 | "최윤식 박사가 눈여겨 본 사건(event, 事件):" |
| 서명 | "미래학자 최윤식 박사" (우측 정렬) |

## 토픽 명명 규칙

대괄호 `[]` 안에 **분석 대상 + 핵심 키워드**를 포함:

- `[트럼프 행정부의 이란 공격 가능성 및 글로벌 파급효과 분석]`
- `[태자당 동향 분석: 장유샤 숙청과 시진핑 1인 지배 체제의 구조적 위기]`
- `[AI 발 사모신용(Private Credit) 위기와 제2의 금융위기 가능성 분석]`
- `[비트코인 가격 하락의 다각적 원인 분석]`

## 번호 매김 규칙

- 토픽 내 소섹션: `### 1.`, `### 2.`, ... (아라비아 숫자)
- 소섹션 내 소제목: **볼드체** (번호 없음)
- 각 토픽은 1부터 다시 시작

## 인라인 포맷팅 규칙

| 요소 | 포맷 | 예시 |
|------|------|------|
| 전문용어 영문 병기 | 한국어(English) | 사모신용(Private Credit) |
| 한자 병기 | 한국어(漢字) | 사궁(死宮), 사건(事件) |
| 강조 소제목 | **볼드:** | **국내 정치적 사궁(死宮):** |
| 인용/발언 | 작은따옴표 | '90% 전쟁 확률' |
| 직접 인용 | 큰따옴표 | "외교 외의 선택지" |
| 기관/매체명 | 그대로 | 악시오스, OCC(통화감독청) |
