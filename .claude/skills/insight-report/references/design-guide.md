# 디자인 가이드 (Design Guide)

> 최윤식 박사의 통찰보고서 PDF 원본에서 추출한 시각적 디자인 패턴.
> PDF/Word 변환 시 이 가이드를 정확히 따른다.

## 1. 페이지 레이아웃

### 용지 및 여백

| 속성 | 값 |
|------|-----|
| 용지 | A4 (210 x 297 mm) |
| 상단 여백 | 2.5cm |
| 하단 여백 | 2.0cm |
| 좌우 여백 | 2.5cm |
| 본문 영역 | 약 16cm 폭 |

### 푸터

- **좌하단**: (비어있음)
- **중앙 하단**: `www.afhi.org` (9pt, 회색)
- **우하단**: 페이지 번호 (9pt)

## 2. 배너 영역 (1페이지 상단)

### 구조

```
┌─────────────────────────────────────────┐
│  ──────────────────────────────────────  │  ← 상단 가로선 (진한 회색, 1pt)
│                                         │
│    미래학자 최윤식 박사의 미래통찰          │  ← 혼합 크기 타이포그래피
│                                         │
│  ──────────────────────────────────────  │  ← 하단 가로선 (진한 회색, 1pt)
└─────────────────────────────────────────┘
```

### 배너 타이포그래피 (핵심)

배너 제목은 **균일한 크기가 아닌 혼합 크기**로 구성된다:

| 텍스트 | 크기 | 굵기 | 역할 |
|--------|------|------|------|
| "미래학자" | 14pt | Regular | 수식어 (작게) |
| "최윤식" | 24pt | Bold | 저자명 (크게, 강조) |
| "박사의" | 14pt | Regular | 수식어 (작게) |
| "미래통찰" | 24pt | Bold | 보고서명 (크게, 강조) |

렌더링 결과:

```
미래학자 최윤식 박사의 미래통찰
         ^^^^           ^^^^^^^^
         24pt Bold      24pt Bold
^^^^^^^       ^^^^^
14pt          14pt
```

- 전체 배너는 **중앙 정렬**
- 상하 가로선은 **전체 본문 폭**을 차지
- 가로선 색상: `#666666` (진한 회색) 또는 `#333333`
- 가로선 두께: 1pt solid

## 3. 밑줄 강조 패턴

원본 PDF에서 밑줄은 **하이퍼링크가 아닌 시각적 강조**로 사용된다:

### 밑줄 적용 대상

| 대상 | 예시 | 의미 |
|------|------|------|
| 날짜 | <u>2026년 2월 27일</u>, 존경하는 독자들에게... | 발행일 강조 |
| 저자 역량 핵심 구간 | 필자는 <u>**2012년부터**</u> 신흥국 금융위기... | 트랙레코드 시작점 강조 |
| 메인 타이틀 | <u>Watching Crisis: 몇 가지 단상들</u> | 보고서 제목 강조 |

### 밑줄 스타일

- CSS: `text-decoration: underline`
- Word: 표준 밑줄
- 색상: 본문 텍스트와 동일 (검정)
- 밑줄 + **볼드** 조합 사용 가능 ("**2012년부터**"는 볼드+밑줄)

## 4. 텍스트 계층 (Typography Hierarchy)

### 폰트

| 용도 | 폰트 (Mac) | 폰트 (Windows) | 대체 |
|------|-----------|---------------|------|
| 본문 | Apple SD Gothic Neo | 맑은 고딕 (Malgun Gothic) | Noto Sans KR |
| 영문 혼용 | 시스템 산세리프 | 시스템 산세리프 | Arial |

### 크기 계층

| 요소 | 크기 | 굵기 | 색상 | 간격 |
|------|------|------|------|------|
| 배너 강조 (최윤식, 미래통찰) | 24pt | Bold | #000 | — |
| 배너 일반 (미래학자, 박사의) | 14pt | Regular | #000 | — |
| 메인 타이틀 (H1) | 18pt | Bold | #000 | 상단 24px |
| 토픽 제목 [대괄호] (H2) | 16pt | Bold | #000 | 상단 32px |
| 소섹션 번호 (H3) | 14pt | Regular | #000 | 상단 20px |
| 볼드 소제목 | 11pt | **Bold** | #000 | 상단 12px |
| 본문 | 11pt | Regular | #222 | 행간 1.7 |
| 발간 기준 | 9.5pt | Regular | #444 | — |
| 푸터 | 9pt | Regular | #888 | — |

### 소섹션 제목 패턴

원본 PDF의 소섹션 번호(`1.`, `2.`, ...)는 **H3이 아닌 중간 크기 텍스트**:

```
1. 중동 정세의 급격한 변화와 군사적 긴장 현황     ← 14pt, 기본 굵기
                                                      (H3 마크다운 ### 대응)
```

토픽 제목 `[대괄호]`는 더 큰 별도 블록:

```
[트럼프 행정부의 이란 공격 가능성 및 글로벌 파급효과 분석]   ← 16pt Bold
                                                              (H2 마크다운 ## 대응)
```

## 5. 볼드 소제목 레이아웃

### 핵심 규칙: 콜론 뒤 같은 줄 시작

원본 PDF에서 볼드 소제목은 **콜론(:) 뒤에 줄바꿈 없이** 바로 본문이 시작된다:

```
국내 정치적 사궁(死宮):
지지율 급락과 악재: 앱스타인 사건 명단에 하워드 러트닉 등 측근이 포함된 점과
아이스(ICE)의 시민권자 오인 사살 사건으로 마가(MAGA) 세력의 실망감이 커지며
비호감도가 급증하고 있다.
```

- 볼드 소제목 자체는 별도 줄 (본문과 분리)
- 콜론(`:`) 뒤에 **줄바꿈**
- 다음 줄부터 일반 본문이 이어짐
- 소제목과 다음 소제목 사이: 본문 1-2 문단

### Markdown 대응

```markdown
**국내 정치적 사궁(死宮):**
지지율 급락과 악재: 앱스타인 사건 명단에...
```

HTML 렌더링 시 `<strong>` 태그 뒤 `<br/>` 또는 별도 `<p>`:

```html
<p><strong>국내 정치적 사궁(死宮):</strong><br/>
지지율 급락과 악재: 앱스타인 사건 명단에...</p>
```

## 6. 표 디자인

### 원본 PDF 표 스타일

원본 PDF의 표는 **최소주의적(Minimalist)** 디자인:

| 속성 | 값 |
|------|-----|
| 테두리 | 가로선만 (상단, 헤더 하단, 하단) — 세로선 없음 또는 최소 |
| 헤더 배경 | 없음 또는 매우 연한 회색 (#F5F5F5) |
| 셀 패딩 | 6px 상하, 8px 좌우 |
| 폰트 크기 | 10pt (본문보다 1pt 작음) |
| 정렬 | 좌측 정렬 기본 |
| 열 폭 | 첫 열 좁게 (라벨), 두 번째 열 넓게 (설명) |

### p.13 표 특수 패턴 (법안 비교표)

법안 비교 등 **정보 밀도 높은 표**:

```
┌──────────┬───────────────────────────────────────────┐
│ 법안명     │ 핵심 내용 및 현재 상태 (소스 기준)            │
├──────────┼───────────────────────────────────────────┤
│ 루미스법   │ 5년간 비트코인 100만 개 전략 비축 추진.       │
│ (Lummis  │ 현재 국가 부채 문제와 국회 통과 난항으로       │
│  Bill)    │ 동력을 상실한 상태.                           │
├──────────┼───────────────────────────────────────────┤
│ 클래리티법 │ **제404조(이자 지급 금지)**를 둘러싼 은행권과 │
│ (Clarity │ 업계 간 갈등으로 상원 심의 중단.               │
│  Act)    │ 현재 가상자산 입법의 최대 병목 구간.            │
└──────────┴───────────────────────────────────────────┘
```

특징:
- 셀 내부에 **볼드** 강조 사용 (`**제404조**`)
- 빈 셀 없이 모든 칸에 상세 설명
- 영문 원어를 괄호 안에 작은 글씨로 포함

## 7. 색상 팔레트

### 최소 색상 원칙

원본 PDF는 **거의 흑백**이며, 색상 사용이 극도로 절제됨:

| 용도 | 색상 | 코드 |
|------|------|------|
| 본문 텍스트 | 검정/짙은 회색 | #222222 |
| 볼드 텍스트 | 검정 | #000000 |
| 배너 가로선 | 진한 회색 | #333333 |
| 발간 기준 텍스트 | 중간 회색 | #444444 |
| 푸터 (URL, 페이지) | 연한 회색 | #888888 |
| 하이퍼링크 | 파랑 | #0000EE (밑줄) |
| 표 테두리 | 연한 회색 | #CCCCCC |

**사용하지 않는 것:**
- 배경색 없음 (순수 흰 배경)
- 표 헤더 음영 없음 (또는 극히 연한 회색만)
- 색상 강조 없음 (빨강, 파랑 등 사용 안 함)
- 박스/카드 UI 없음

## 8. 구분선 패턴

### 토픽 간 구분

토픽 사이에는 **넓은 수직 여백** (약 3-4 줄 높이)으로 구분. 명시적 가로선 없음.

```
...이란에 대한 면밀한 모니터링이 요됨.

                                          ← 약 3-4줄 높이 빈 공간


[태자당 동향 분석: 장유샤 숙청과 시진핑 1인 지배 체제의 구조적 위기]
```

### 헤더 영역 구분

배너 아래, 메인 타이틀 위에는 **가로선 + 여백**으로 구분:

```
발간 기준: 본 보고서는...

───────────────────────        ← 가로선 (CSS: hr, 또는 border-top)

Watching Crisis: 몇 가지 단상들
```

### Markdown `---` → PDF 렌더링

마크다운의 `---`는 PDF에서 **가느다란 가로선** (1px, #CCCCCC)으로 렌더링:
- 배너 구분선보다 **가늘고 연한** 색상
- 토픽 간 시각적 분리 역할

## 9. 서명 영역

### 마지막 페이지

```
                                          미래학자 최윤식 박사
```

- **우측 정렬**
- 본문과 동일 폰트, 11pt
- 마지막 본문에서 **2-3줄 간격** 후
- 볼드 아님 (일반 굵기)

## 10. PDF 변환 CSS 사양

위 디자인 패턴을 weasyprint/CSS로 구현하기 위한 핵심 CSS:

```css
@page {
    size: A4;
    margin: 2.5cm 2.5cm 2cm 2.5cm;
    @bottom-center {
        content: "www.afhi.org";
        font-size: 9pt;
        color: #888;
    }
    @bottom-right {
        content: counter(page);
        font-size: 9pt;
        color: #888;
    }
}

body {
    font-family: 'Apple SD Gothic Neo', 'Malgun Gothic', 'Noto Sans KR', sans-serif;
    font-size: 11pt;
    line-height: 1.7;
    color: #222;
}

/* 배너 영역 */
.banner {
    text-align: center;
    border-top: 1pt solid #333;
    border-bottom: 1pt solid #333;
    padding: 16px 0;
    margin-bottom: 24px;
}
.banner .small { font-size: 14pt; }
.banner .large { font-size: 24pt; font-weight: bold; }

/* 밑줄 강조 */
.underline { text-decoration: underline; }

/* 제목 계층 */
h1 {
    font-size: 18pt;
    font-weight: bold;
    text-decoration: underline;
    margin-top: 24px;
    border: none;  /* 가로선 아님 */
}

h2 {
    font-size: 16pt;
    font-weight: bold;
    margin-top: 32px;
}

h3 {
    font-size: 14pt;
    font-weight: normal;  /* 원본은 기본 굵기 */
    margin-top: 20px;
}

/* 볼드 소제목 */
strong {
    color: #000;
    font-weight: bold;
}

/* 표 */
table {
    border-collapse: collapse;
    width: 100%;
    margin: 12px 0;
    font-size: 10pt;
}
th, td {
    padding: 6px 8px;
    border-bottom: 1px solid #ccc;
    text-align: left;
    vertical-align: top;
}
th {
    border-top: 2px solid #333;
    border-bottom: 1px solid #333;
    font-weight: bold;
    background: none;  /* 원본: 배경색 없음 */
}
tr:last-child td {
    border-bottom: 2px solid #333;
}

/* 구분선 */
hr {
    border: none;
    border-top: 1px solid #ccc;
    margin: 24px 0;
}

/* 발간 기준 */
.publication-note {
    font-size: 9.5pt;
    color: #444;
    line-height: 1.5;
}

/* 서명 */
.signature {
    text-align: right;
    margin-top: 48px;
    font-size: 11pt;
}

/* 푸터 URL */
.footer-url {
    text-align: center;
    font-size: 9pt;
    color: #888;
}
```

## 11. 현재 스킬 PDF 변환과의 차이점

| 항목 | 현재 구현 | 원본 PDF | 수정 필요 |
|------|---------|---------|---------|
| 배너 타이포그래피 | 균일 크기 | **혼합 크기** (14pt/24pt) | **예** |
| 배너 가로선 | `━━━` 문자 또는 빨간 border | 진한 회색 1pt solid | **예** |
| 밑줄 강조 | 없음 | 날짜·제목·역량 밑줄 | **예** |
| 표 헤더 배경 | 파란 음영 (#D9E2F3) | 배경 없음 | **예** |
| 표 테두리 | 전체 테두리 | 가로선만 (세로선 최소) | **예** |
| H3 굵기 | Bold | Regular (기본 굵기) | **예** |
| 색상 | 빨간 가로선, 파란 제목 | **거의 흑백** | **예** |
| 구분선 | 빨간 2px | 연한 회색 1px | **예** |
| 푸터 | 페이지 번호만 | URL + 페이지 번호 | **예** |
| H1 스타일 | border-bottom 빨강 | **밑줄 (underline)** | **예** |
